# FilePilot 开发文档

> 版本：1.3.0
> 更新时间：2026-07-05

---

## 一、项目概述

FilePilot 是一款 Windows 桌面文件索引与预览工具，支持快速搜索本地文件、预览多种格式文件（Office、PDF、Markdown、图片等），并提供浏览器式标签页交互体验。

## 二、技术架构

### 2.1 技术栈

| 模块 | 技术选型 | 说明 |
|------|----------|------|
| 前端框架 | Vue 3 + TypeScript | 响应式 UI，组件化开发 |
| UI 组件库 | Vuetify 4 | Material Design 风格 |
| 状态管理 | Pinia | Vue 3 官方状态管理 |

## 三、核心功能实现

### 3.1 数据库设计

使用 SQLite（sql.js），单表结构：

```sql
-- 文件信息表
CREATE TABLE files (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    root_path TEXT NOT NULL,
    relative_path TEXT NOT NULL,
    file_name TEXT NOT NULL,
    file_size INTEGER,
    modified_time INTEGER,
    file_ext TEXT,
    is_indexed INTEGER DEFAULT 0,
    created_at TEXT DEFAULT (datetime('now', 'localtime')),
    updated_at TEXT DEFAULT (datetime('now', 'localtime')),
    UNIQUE(root_path, relative_path)
);

-- 根目录表
CREATE TABLE root_directories (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    path TEXT UNIQUE NOT NULL,
    file_count INTEGER DEFAULT 0,
    total_size INTEGER DEFAULT 0,
    last_indexed TEXT,
    created_at TEXT DEFAULT (datetime('now', 'localtime')),
    updated_at TEXT DEFAULT (datetime('now', 'localtime'))
);

-- 最近文件表
CREATE TABLE recent_files (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    root_path TEXT NOT NULL,
    rel_path TEXT NOT NULL,
    file_name TEXT NOT NULL,
    accessed_at TEXT DEFAULT (datetime('now', 'localtime'))
);

-- 设置表
CREATE TABLE settings (
    key TEXT PRIMARY KEY,
    value TEXT
);
```

### 3.2 文件索引

- 递归遍历目录，跳过隐藏文件夹（`.`开头）
- 存储文件元数据（路径、大小、修改时间、扩展名）
- 支持增量更新（比对 modified_time）
- 索引完成后清理已删除文件记录

### 3.3 文件预览

使用 Worker 线程池处理文件转换，避免阻塞主线程：

| 文件类型 | 转换方式 | 输出格式 |
|----------|----------|----------|
| DOCX/DOC | mammoth | HTML（保留格式） |
| XLSX/XLS | xlsx | HTML 表格 |
| PDF | pdf-parse | 提取文本 |
| MD | marked | 渲染 HTML |
| TXT/代码 | 直接读取 | 纯文本 |
| 图片 | base64 内嵌 | HTML img |
| 其他 | 二进制检测 | 信息提示 |

### 3.4 线程池管理

```javascript
class WorkerPool {
  constructor(maxWorkers) // 最大线程数 = CPU核心数 - 1
  execute(taskData, onProgress) // 执行任务，支持进度回调
  checkStuckWorkers() // 定期检查卡死线程（60秒超时）
  processQueue() // 处理等待队列
  shutdown() // 关闭所有线程
}
```

### 3.5 标签页系统

- 浏览器式标签页，支持关闭和切换
- 点击文件在新标签页打开预览
- 同一文件只打开一个标签页（按路径去重）
- 每个标签页独立维护浏览状态

### 3.6 大纲导航

- 从 h1-h6 标签提取标题生成大纲
- DOCX：从 Word 标题样式提取
- XLSX：使用工作表名称作为大纲
- MD：从 Markdown 标题语法提取
- 点击大纲项平滑滚动到对应位置
- 大纲侧边栏可折叠，独立于内容滚动

## 四、IPC 通信接口

### 4.1 文件夹管理

| 接口 | 参数 | 说明 |
|------|------|------|
| `folders:list` | - | 获取所有文件夹 |
| `folders:browse` | - | 打开系统文件夹选择器 |
| `folders:add` | path | 添加文件夹并开始索引 |
| `folders:remove` | id | 删除文件夹及其索引 |
| `folders:reindex` | id | 重新索引文件夹 |

### 4.2 文件操作

| 接口 | 参数 | 说明 |
|------|------|------|
| `search` | keyword, limit | 搜索文件名 |
| `recent:list` | limit | 获取最近打开文件 |
| `recent:delete` | rootPath, relPath | 删除历史记录 |
| `preview` | id | 预览文件（返回内容+大纲） |
| `open:location` | filePath | 在资源管理器中显示 |
| `open:file` | filePath | 用默认程序打开 |

### 4.3 系统接口

| 接口 | 参数 | 说明 |
|------|------|------|
| `settings:get` | - | 获取设置 |
| `settings:update` | settings | 更新设置 |
| `stats` | - | 获取统计数据 |
| `window:minimize` | - | 最小化窗口 |
| `window:maximize` | - | 最大化/还原窗口 |
| `window:close` | - | 关闭窗口 |
| `get:server-port` | - | 获取本地 HTTP 服务端口 |
| `worker:status` | - | 获取线程池状态 |

## 五、构建与打包

### 5.1 开发环境

```bash
cd frontend
npm install
npm run dev:electron
```

## 六、已知问题与优化方向

### 6.1 已知问题

- 旧版本数据库中的时间是 UTC 格式，新版本已改为本地时间
- 超大文件（>500MB）的文本读取可能较慢

### 6.2 优化方向

- 添加全文搜索（FTS5）
- 支持更多文件格式预览
- 添加文件标签和收藏功能
- 支持多窗口
- 添加快捷键支持
