# FilePilot - 本地文件索引与预览工具

> 版本：1.3.0
> 平台：Windows 10/11

---

## 简介

FilePilot 是一款轻量级的本地文件管理工具，帮助您快速搜索、浏览和预览电脑中的文件。无需打开多个应用程序，一个界面即可查看各种格式的文件内容。

---

## 核心功能

### 🔍 快速搜索

- 输入关键词即可搜索文件名
- 支持模糊匹配
- 搜索结果实时显示

### 📄 多格式预览

| 格式 | 预览方式 |
|------|----------|
| Word (DOCX/DOC) | 转换为 HTML，保留格式和样式 |
| Excel (XLSX/XLS) | 转换为 HTML 表格，支持多工作表 |
| PDF | 提取文本内容显示 |
| Markdown | 渲染为富文本，支持标题、列表、代码块 |
| TXT/代码 | 语法高亮显示 |
| 图片 (JPG/PNG/GIF/BMP/WebP/SVG) | 直接显示图片 |
| 其他格式 | 显示文件信息，支持用默认程序打开 |

### 🏷️ 浏览器式标签页

- 点击文件在新标签页打开预览
- 支持多标签页同时查看
- 标签页可关闭和切换
- 同一文件不会重复打开

### 📋 大纲导航

- Word、Excel、Markdown 文件自动生成大纲
- 点击大纲项快速跳转到对应位置
- 大纲侧边栏可折叠
- 适合查看长文档

### 📁 文件夹管理

- 添加本地文件夹建立索引
- 支持系统文件夹选择器
- 实时显示索引进度
- 支持重新索引和移除

### ⚙️ 个性化设置

| 设置项 | 说明 |
|--------|------|
| 主题模式 | 浅色 / 深色 |
| 大文件阈值 | 跳过超过指定大小的文件 |
| 索引隐藏文件夹 | 是否索引以 `.` 开头的文件夹 |
| 开机自启动 | 系统启动时自动运行 |

### 📊 文件统计

- 文件类型分布饼图
- 各类型文件数量和大小
- 索引文件夹总数

### 🖱️ 右键菜单

- 搜索结果：在新标签页打开、复制路径、打开所在位置
- 历史记录：打开、复制路径、从记录中移除
- 文件夹卡片：重新索引、复制路径、移除

### ℹ️ 关于页面

- 显示软件版本信息
- 开发者信息及社交媒体链接
- 技术栈和功能特性
- 免责声明和版权信息

## 使用方法

### 安装

1. 下载 `FilePilot Setup 1.3.0.exe`
2. 双击运行安装程序
3. 按提示完成安装
4. 桌面会出现 FilePilot 快捷方式

### 添加文件夹

1. 点击左侧菜单「文件夹」
2. 点击「添加文件夹」卡片
3. 在弹出的对话框中选择文件夹
4. 等待索引完成

### 搜索文件

1. 在首页搜索框输入关键词
2. 按回车或点击「搜索」按钮
3. 点击搜索结果即可预览

### 预览文件

- 点击文件列表中的文件，自动在新标签页打开预览
- 支持 Word、Excel、PDF、Markdown 等格式
- 点击「打开文件」可用系统默认程序打开
- 点击「复制路径」可复制文件完整路径

### 使用大纲

1. 打开 Word、Excel 或 Markdown 文件
2. 右侧会自动显示大纲侧边栏
3. 点击大纲项可跳转到对应位置
4. 点击右上角箭头可折叠大纲

## 常见问题

### Q: 为什么某些文件无法预览？

A: 目前支持 Word、Excel、PDF、Markdown、TXT、常见图片格式的预览。其他格式（如 PSD、XCF、视频等）会显示文件信息，可点击「打开文件」使用系统默认程序查看。

### Q: 索引速度很慢怎么办？

A: 索引速度取决于文件数量和磁盘性能。首次索引需要遍历所有文件，后续索引只处理变更文件，会快很多。

### Q: 如何删除索引数据？

A: 在文件夹管理页面，右键点击文件夹卡片，选择「移除」即可删除该文件夹的所有索引数据。

### Q: 支持网络驱动器吗？

A: 支持，只要网络驱动器已映射为本地驱动器号即可添加。
